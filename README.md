 DWLR Water Monitoring App

 About
An Android application that visualizes **DWLR (Digital Water Level Recorder) station data** across India using an interactive map.  
Users can tap on any station to view real-time and historical water statistics, presented through an intuitive and color-coded interface.

 Features
-  Interactive **Map of India** with DWLR station markers
-  Tap on a station to view:
  - Water level data
  - Station-specific statistics
-  Color-coded stations for quick visual understanding
-  Clear and user-friendly data presentation
-  Smooth map interaction and fast data loading

 Tech Stack
- Android Studio
- Kotlin / Java
- Google Maps API
- DWLR station dataset
- Material UI Components


 How It Works
1. DWLR station data is fetched and processed.
2. Stations are plotted as markers on the India map.
3. Each marker is color-coded based on water level thresholds.
4. On tapping a station, detailed water statistics are displayed.


 Future Improvements
- Real-time data updates
- Graphical water level trends
- Offline data caching
- Alert system for critical water levels
